# Typography

## Typeface Direction

- `font.display`: editorial serif with warmth, contrast, and sculptural elegance.
- `font.body`: modern sans-serif with calm geometry and excellent readability.

## Working Font Pairing

- Display working direction: `Cormorant Garamond` or an equivalent refined editorial serif.
- Body and UI working direction: `Instrument Sans` or an equivalent clean sans-serif.

These are working system choices for implementation planning, not a claim about the client’s final brand assets.

## Usage Rules

### Display Serif

- Hero headlines
- Major section headings
- Short pull-quote or value statements if later approved

### Sans Serif

- Navigation
- Body text
- Cards
- Forms
- Breadcrumbs
- Buttons
- Specs and metadata

## Scale

| Style | Mobile | Desktop | Use |
| --- | --- | --- | --- |
| Hero display | `clamp(2.5rem, 8vw, 5.5rem)` | `clamp(3.5rem, 6vw, 6rem)` | Homepage hero |
| Page heading | `2rem` | `3rem` | Interior page titles |
| Section heading | `1.5rem` | `2.25rem` | Section intros |
| Card heading | `1.125rem` | `1.25rem` | Product and collection cards |
| Intro body | `1.0625rem` | `1.1875rem` | Short editorial intros |
| Body | `1rem` | `1rem` | Standard copy |
| Secondary body | `0.9375rem` | `0.9375rem` | Meta and helper copy |
| Micro/meta | `0.8125rem` | `0.875rem` | Labels, breadcrumbs, tags |

## Line Height Rules

- Display serif: tight but breathable, about `1.02` to `1.12` depending on size.
- Section and page headings: about `1.15` to `1.2`.
- Body text: about `1.65`.
- Labels and metadata: about `1.4` to `1.5`.

## Measure And Readability

- Long-form text should stay around `60` to `72` characters per line.
- Intro paragraphs may run slightly wider but should stay below `78` characters.
- Product specification lines should stay compact and scan-friendly.

## Styling Rules

- Use high-contrast serif display moments sparingly so they retain value.
- Avoid all-caps for long navigation or body blocks.
- Use letter spacing lightly for metadata and labels only.
- Do not rely on ultra-light weights for primary reading text.

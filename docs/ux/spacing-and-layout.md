# Spacing And Layout

## Layout Principles

- Use wide outer breathing room and disciplined inner rhythm.
- Let imagery and headings land in clearly framed blocks.
- Avoid cramped card grids and overpacked text columns.

## Max Widths

| Content Type | Max Width |
| --- | --- |
| Reading column | `42rem` to `46rem` |
| Narrow editorial intro | `52rem` |
| Standard page container | `72rem` |
| Wide catalogue container | `80rem` |
| Large desktop hero container | `88rem` |
| Ultra-wide cap | `96rem` |

## Horizontal Gutters

| Viewport | Gutter |
| --- | --- |
| Small mobile | `1rem` |
| Large mobile | `1.25rem` |
| Tablet | `1.5rem` |
| Laptop | `2rem` |
| Desktop | `3rem` |
| Large desktop | `4rem` |

## Vertical Rhythm

| Element | Space Rule |
| --- | --- |
| Section to section on mobile | `4rem` to `5rem` |
| Section to section on desktop | `6rem` to `8rem` |
| Heading to intro text | `0.75rem` to `1rem` |
| Intro to content grid | `1.5rem` to `2rem` |
| Card content stack | `0.5rem` to `0.75rem` |
| Form field stack | `1rem` to `1.25rem` |

## Grid Rules

### Collections Index

- Mobile: `1` column
- Tablet portrait: `2` columns
- Desktop: `3` columns
- Large desktop: `4` columns only if imagery still feels generous

### Products Index

- Small mobile: `1` to `2` columns depending on card width
- Tablet: `2` to `3` columns
- Laptop: `3` columns
- Desktop: `4` columns

### Related Products

- Mobile: horizontal scroll or `1` column stack depending on final density
- Tablet and up: `2` to `4` cards

## Image Ratio Guidance

| Component | Preferred Ratio |
| --- | --- |
| Hero | cinematic landscape, about `16:9` to `3:2` |
| Collection card | vertical editorial ratio, about `4:5` |
| Product card | stable portrait or near-square ratio, about `4:5` or `1:1.15` |
| Product gallery primary | generous landscape or natural ratio with controlled frame |
| Product gallery thumbnails | square or uniform crop |

## Page-Specific Layout Notes

- Homepage should alternate text-led and image-led rhythms to avoid one long repeated pattern.
- Product detail should prioritize immediate image visibility, core specs, and inquiry access above the fold.
- About and Craftsmanship pages should use narrower text columns and more editorial pacing than the catalogue.
- Legal pages should collapse to simple reading layouts with minimal decorative treatment.

# Editorial Guidelines

## Editorial Principles

- Tone should be warm, refined, human, craft-focused, and concise.
- Copy should feel premium and calm rather than sales-heavy or corporate.
- Every public statement must be verifiable through client-supplied information.
- No invented testimonials, awards, manufacturing claims, heritage claims, or statistics.

## Voice Direction

- Use plain, elegant language.
- Prefer specific material and craftsmanship details over generic adjectives.
- Write for visitors who are visually driven but still need practical product understanding.
- Keep calls to action direct and respectful.

## Product Description Structure

- Product name
- One short descriptive summary for quick scanning
- A fuller paragraph explaining the product's character, placement, or decorative role using approved facts only
- Structured specifications for dimensions, material, finish, category, and collection context where applicable
- Inquiry-oriented close that invites conversation without implying direct purchase flow

## Collection Copy Structure

- Collection name
- Brief framing sentence on the collection theme or style
- Optional extended description if the client provides enough editorial context

## About And Craftsmanship Copy Structure

- Start with a concise factual introduction.
- Break longer content into short editorial sections with clear headings.
- Use process language only where the client has supplied accurate detail.
- Avoid romanticized claims that cannot be verified.

## CTA Writing Rules

- Preferred action patterns: Explore Collections, View Products, Send Inquiry, Contact Us, Chat on WhatsApp.
- Avoid transactional labels such as Buy Now, Add to Cart, Checkout, or Order Today.
- CTA copy should match the actual destination and behavior.

## Title Length Guidance

- Navigation labels: usually 1 to 3 words.
- Page headings: concise, usually under 70 characters.
- Product names: determined by client data, but should avoid unnecessary filler terms when editing is allowed.

## Meta Title Guidance

- Target length: about 50 to 60 characters when practical.
- Include brand context when useful, especially on brand and static pages.
- Keep each title unique.

## Meta Description Guidance

- Target length: about 140 to 160 characters when practical.
- Summarize the page clearly without keyword stuffing.
- Avoid claims that cannot be verified.

## Image Alt-Text Guidance

- Alt text should describe the image's meaningful content in context.
- Product alt text should identify the product and the visible view or defining visual detail.
- Decorative images should use empty alt text in implementation if they add no informational value.

## Content Fallback Rules

- If required approved copy is missing, do not invent it.
- If a section is optional and lacks approved content, hide it.
- If legal text is not approved, do not publish the legal page.
- If contact details are not approved, keep contact shortcuts out of public output until supplied.

## Restricted Content

- No fabricated reviews or testimonials.
- No fabricated claims about origin, process, scale, awards, or years in business.
- No unapproved guarantees, delivery promises, pricing claims, or stock claims.
